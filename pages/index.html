what is .apf

what it is

APF stands for Aperture Image Format, presented as an in-universe Aperture Science format from Valve's Portal.

how compression works

A 1-bit black-and-white bitmap is stored and compressed with run-length encoding, writing alternating black and white run counts in sequence.

It also supports a line-skip pass order so rows are traversed in a stepped pattern instead of plain top-to-bottom, then decoded back with the same pass logic.

limitations

Images are converted to exactly 320x200 and pure black/white only. No grayscale and no color are preserved.

Very short details can disappear after resize + 1-bit conversion, and output PNG downloads keep APF dimensions and aspect ratio.
